The Problem

  • Limited Action: Current AI assistants can answer questions but cannot navigate complex game engines or OS tasks.
  • Fragmented Workflow: Game devs switch between Blender, Unity, Discord, and Browsers constantly.
  • No Autonomous Agents: You have to play alone or script complex bots manually.
  • Cloud Dependency: High latency cloud AI is too slow for real-time gaming or asset generation.

The Solution

Multivac AI: Gaming Edition

A sophisticated local entity evolving from Asimov's supercomputer. It combines Gemini 2.5 Native Audio with precise game control and vision.

Universal Game Master: Whether managing your RGB environment, generating assets on the fly, or playing alongside you as an AI Agent, Multivac executes complex gaming workflows.

Complete System Control

Multivac Voice

Real-time, low-latency command channel. Shout commands in-game without alt-tabbing.

Input Control

System-wide keyboard and mouse automation. It can execute complex combos or manage inventory.

Window Mgmt

Launch games, resize streams, and focus Discord via natural language commands.

Physical & Digital Bridging

Asset Gen

Generate 3D game assets, textures, and simple levels via voice prompts directly to STL/OBJ.

Vision & Gesture

"Minority Report" style HUD control via webcam. Pinch to mute, swipe to switch scenes.

AI Game Agent

NEW: Deploy an autonomous AI agent to test levels or play coop as a smart NPC.

Architecture

  • AI Engine: Gemini 2.5 Native Audio + Vision
  • Backend: Python 3.11, FastAPI, Socket.IO
  • Game Integration: Direct hooks into OS inputs and window management.
  • Agents: Modular Python agents for Assets (build123d), Web, and Gaming Logic.
